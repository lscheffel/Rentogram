Referência da API

Base URL: http://localhost:3000/api

Propriedades

GET /properties

GET Lista todas as propriedades

Resposta: 200 OK - Array de objetos Property

GET /properties/:id

GET Obtém propriedade por ID

Parâmetros: id (integer)

Resposta: 200 OK - Objeto Property | 404 Not Found

POST /properties

POST Cria nova propriedade

Corpo:

{
  "title": "string",
  "description": "string",
  "address": "string",
  "price_per_night": number,
  "bedrooms": integer,
  "bathrooms": integer,
  "max_guests": integer,
  "amenities": "string",
  "image_url": "string"
}

Resposta: 201 Created - Objeto Property criado

PUT /properties/:id

PUT Atualiza propriedade

Parâmetros: id (integer)

Corpo: Mesmo que POST

Resposta: 200 OK - Objeto Property atualizado

DELETE /properties/:id

DELETE Exclui propriedade

Parâmetros: id (integer)

Resposta: 204 No Content

Reservas

GET /reservations

GET Lista todas as reservas

Resposta: 200 OK - Array de objetos Reservation

GET /reservations/:id

GET Obtém reserva por ID

Parâmetros: id (integer)

Resposta: 200 OK - Objeto Reservation

GET /reservations/property/:id

GET Lista reservas por propriedade

Parâmetros: id (integer) - ID da propriedade

Resposta: 200 OK - Array de objetos Reservation

POST /reservations

POST Cria nova reserva

Corpo:

{
  "property_id": integer,
  "guest_name": "string",
  "guest_email": "string",
  "check_in_date": "YYYY-MM-DD",
  "check_out_date": "YYYY-MM-DD",
  "total_price": number
}

Resposta: 201 Created - Objeto Reservation criado

PUT /reservations/:id

PUT Atualiza reserva

Parâmetros: id (integer)

Corpo: Mesmo que POST

Resposta: 200 OK - Objeto Reservation atualizado

DELETE /reservations/:id

DELETE Exclui reserva

Parâmetros: id (integer)

Resposta: 204 No Content

Respostas de Erro

  • 400 Bad Request: Dados inválidos ou validação falhou
  • 404 Not Found: Recurso não encontrado
  • 500 Internal Server Error: Erro interno do servidor
