Configuração

Esta seção descreve como configurar o ambiente de desenvolvimento e produção do Rentogram.

Pré-requisitos

  • Node.js versão 14 ou superior
  • npm ou yarn
  • SQLite (incluído automaticamente)

Variáveis de Ambiente

Arquivo .env na raiz do projeto:

PORT=3000
NODE_ENV=development

Variáveis Disponíveis

Variável Descrição Padrão
PORT Porta do servidor backend 3000
NODE_ENV Ambiente de execução development

Configuração do Frontend

Arquivo src/frontend/config.ts:

export const API_ENDPOINTS = {
  PROPERTIES: '/api/properties',
  RESERVATIONS: '/api/reservations'
};

export const API_BASE_URL = 'http://localhost:3000';

Configuração do Backend

Arquivo src/config/config.js:

module.exports = {
  cors: {
    origin: ['http://localhost:5173', 'http://localhost:3000'],
    credentials: true
  }
};

Inicialização do Banco de Dados

O banco SQLite é criado automaticamente na primeira execução. O schema é definido em src/database/database.js.
