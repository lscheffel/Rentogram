Glossário

Termos técnicos e conceitos utilizados no projeto Rentogram.

API RESTful
Interface de programação de aplicações que segue os princípios REST para comunicação entre sistemas.
Backend
Parte do sistema responsável pela lógica de negócio, processamento de dados e comunicação com o banco de dados.
CORS
Cross-Origin Resource Sharing - mecanismo que permite requisições de diferentes origens.
CRUD
Create, Read, Update, Delete - operações básicas de manipulação de dados.
Frontend
Parte do sistema responsável pela interface de usuário e interação com o usuário.
MVC
Model-View-Controller - padrão arquitetural que separa a aplicação em três componentes interconectados.
Node.js
Runtime JavaScript para execução de código JavaScript no lado do servidor.
React
Biblioteca JavaScript para construção de interfaces de usuário.
SQLite
Banco de dados relacional leve e file-based.
TypeScript
Superset de JavaScript que adiciona tipagem estática opcional.
