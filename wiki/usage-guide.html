Guia de Uso

Este guia explica como usar o sistema Rentogram para gerenciar propriedades e reservas.

Como Começar

  1. Execute o script start.bat para iniciar o sistema.
  2. Acesse o frontend em http://localhost:5173.
  3. Use a barra de navegação para alternar entre Propriedades e Reservas.

Gerenciando Propriedades

Adicionando uma Propriedade

  1. Navegue para a página "Properties".
  2. Clique em "Adicionar Propriedade".
  3. Preencha os campos: título, endereço, preço, etc.
  4. Clique em "Salvar".

Editando uma Propriedade

  1. Na lista de propriedades, clique em "Editar" na propriedade desejada.
  2. Modifique os campos necessários.
  3. Clique em "Salvar".

Gerenciando Reservas

Criando uma Reserva

  1. Navegue para a página "Reservations".
  2. Clique em "Adicionar Reserva".
  3. Selecione uma propriedade existente.
  4. Preencha informações do hóspede e datas.
  5. Clique em "Salvar".

Dashboard

A página inicial mostra um dashboard com estatísticas básicas e visualização de calendário.
